#koa-router-joi

Joi Validation and Documentation for koa-router.

#### Features:

- Optional input/output validation using [joi][]
- Automatic Documentation in Redoc/Swagger
- built on the great [koa-router][]
- string path support
- meta data support

#### Node compatibility

NodeJS `>= 8.3` is required.
